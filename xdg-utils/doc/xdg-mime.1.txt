XDG-MIME(1)
===========


NAME
----
xdg-mime - manages MIME settings


SYNOPSIS
--------
'xdg-mime' [--info FILE] [--defapp FILE]


DESCRIPTION
-----------
The xdg-mime(1) command wraps desktop environment specific MIME type handling
facilities.

*TODO*: capabilities overview


OPTIONS
-------
--info FILE::
    Prints the given file's MIME type

--defapp FILE::
    Prints the filename of the default handler application' .desktop file,
    also known as the "desktop file name"


EXAMPLES
--------

----
xdg-mime --info /tmp/foobar.png
----
    Prints the MIME type of the file /tmp/foobar.png

----
xdg-mime --defapp /tmp/foobar.png
----
    Prints the name of the .desktop file of the application which is
    registered to open files of the MIME type /tmp/foobar.png has
