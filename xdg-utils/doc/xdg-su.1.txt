XDG-SU(1)
===========


NAME
----
xdg-su - executes a given command as the system's super user


SYNOPSIS
--------
'xdg-su' COMMAND


DESCRIPTION
-----------
The xdg-su(1) command wraps desktop environment specific facilities for gaining
elevated user rights (i.e. becoming root).


EXAMPLES
--------

----
xdg-su xdg-menu --root create <my-global-desktop-file>
----
    Runs the command xdg-menu with root permissions. All parameters following
    the command are passed to the command upon execution
